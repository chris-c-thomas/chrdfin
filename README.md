# chrdfin

A personal financial intelligence desktop application. Multi-instance portfolio
tracking, backtesting, Monte Carlo simulation, optimization, calculators,
market data, news, calendars, and a bundled reference library for financial
topics. A trading roadmap (paper, live, bot) is planned for after v1.0.

Built as a [Tauri v2](https://v2.tauri.app/) native desktop app with a Vite +
React 19 frontend and a Rust computation backend backed by embedded
[DuckDB](https://duckdb.org/).

## Status

Pre-1.0, in active development. Single maintainer. Not yet accepting external
contributions; not yet distributing installers. See [`CHANGELOG.md`](./CHANGELOG.md)
for what's shipped and [`docs/technical-blueprint.md`](./docs/technical-blueprint.md)
for the architecture.
